Class: TriggersDropdown

Scenarii.TriggersDropdown

This React component shows a dropdown with available scenarii elements of 'trigger' type. This is use to choose an existing trigger (that is public, so not inside another scenarii element like a procedure) or to create a new one on the fly (optional). You can filter triggers you want with a type filter as a callback, select default element, etc...

Example

Example of an TriggersDropdown. Use it in React JSX syntax

<TriggersDropdown defaultTriggerId={this.state.triggerId} onChange={this.triggerChanged.bind(this)}
  theme={theme} animationLevel={animationLevel} services={services} />

A callback when the user made a choice (in case of "new trigger" choice, called only after the element creation).

An ID, just in order to be able to make a dom query on it...

Optional parent ID element for creation of a "new trigger" case. In that case, the new element will not appear in the main trigger list! None by default.
label string

The label to put above the input, in the user interface.
icon string

The icon to put on the left of the input, in the user interface.
noCreationPanel boolean

True to avoid opening a popin (a modal frame with settings of the new element). False by default.
typeFilter function

A filter to choose what type of new triggers can be created from the dropdown.
instanceFilter function

A filter to choose what existing public triggers should be listed in the dropdown.
