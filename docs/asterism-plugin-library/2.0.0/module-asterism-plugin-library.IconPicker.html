Class: IconPicker

asterism-plugin-library.IconPicker

This React component shows an icon picker in a Modal component. The user can find & select a Material icon from the standard library. The selection is passed through onChange attribute execution, as a String.

Example

Example of a picker inside a button. Use it in React JSX syntax

<Input
    placeholder='My text goes here'
    label='My label'
    value={title} onChange={this.handleEventChange.bind(this)} className='iconPicker'>
  <div>
    <IconPicker
        theme={theme}
        animationLevel={animationLevel}
        defaultIcon='zoom_in'
        onChange={this.handleValueChange.bind(this)}
    />
  </div>
</Input>
