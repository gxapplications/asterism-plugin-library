Class: ActionColorSwitch

ActionColorSwitch

This React component shows a switch to choose one of the theme's buttons colors available in the UI default theme. The component should be used in a form to confiugure a button that can accept a color depending on the user's choice.

Default color is 'secondary', Available colors are: 'primary', 'secondary', 'inconspicuous' Component properties: ActionColorSwitch.propTypes.
