Class: ActionColorSwitch

ActionColorSwitch

This React component shows a switch to choose one of the theme's buttons colors available in the UI default theme. The component should be used in a form to configure a button that can accept a color depending on the user's choice.

Default color is 'secondary'. Available colors are: 'primary', 'secondary', 'inconspicuous'.

Example

Can use it in React JSX syntax

<ActionColorSwitch
    theme={this.props.theme}
    animationLevel={this.props.animationLevel}
    defaultColor="primary"
    onChange={(color) => { this.currentColor = color; }}
/>
