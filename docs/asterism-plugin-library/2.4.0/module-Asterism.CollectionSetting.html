Class: CollectionSetting

Asterism.CollectionSetting

This React component shows a collection of items, most often clickable for edition. At the end of the list, an item can be added to let add more items.

Default header is to have no header (empty string)

Example

Use it in React JSX syntax

<CollectionSetting
    theme={theme}
    animationLevel={animationLevel}
    list={
        elements.map((el) => ({
            title: el.name || 'Unnamed',
            icon: 'favorite',
            onClick: this.configureElement.bind(this, el),
            details: el.details || 'No more details',
            css: 'my-class',
            secondary: {
                icon: 'more_vert',
                onClick: () => {},
                css: 'my-secondary-class'
            }
        }))
    }
    header='My elements list'
    addElement={{
        empty: { title: 'Add your first element here', icon: 'add' },
        trailing: { title: 'Add an element' },
        onClick: this.addElement.bind(this)
    }}
/>
