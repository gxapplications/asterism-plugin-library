The callback method with the pattern once drawn correctly. The pattern is given as parameter, with digits separated by '-'.

The min length for input pattern. If draw is shorten than this number, an error style is shown for 3 seconds.

This allows a pattern to use more than once the same node. Allowing this you can reach a very complicated pattern.
